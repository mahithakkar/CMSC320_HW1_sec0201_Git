Q1: Why might we consider using Git?
Answer: Git is basically a change tracking system that improves collaboration with others and allows the user to rewind if a mistake has been made.

Q2: What field could data science be applied to? Give an Example.
Answer: Data science can be used in healthcare, for example finding patterns in medical records.

Q3: What is the difference between Merging and Rebasing?
Answer: Merge combines branches with a new commit, rebase rewrites history to look cleaner.
